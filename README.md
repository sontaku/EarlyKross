## 서비스

1. **베스트 일레븐 맞추기** - 적중 선수 한 선수당 포인트 획득(포인트는 계정 경험치 Lv++)

2. 리그별, **팀별 게시판**(커뮤니티)

3. **데이터 센터**

   - 선수조회
   - 팀 조회
   - 감독 조회
   - 리그 순위 ( 팀 순위, 선수 개인 순위 )

4. 실시간 중계 채팅방

5. 같이 응원 / 같이 축구

6. 토론

7. (임의)선수비교?팀비교?

8. K리그 뉴스

9. 관리자 페이지

   - 회원관리

     - 블랙리스트
     - 회원조회

   - 시각화 데이터(회원, 게시판 등등)
